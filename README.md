## Get started

- clone the report
- add a file api.js in src/utils folder:

```js
const theMovieDbApiKey = '<InsertTheMovieDbApiKeyHere>';

export {
  theMovieDbApiKey
};
```
- run ```yarn```
- run ```yarn start```
- go on http://localhost:3000/
